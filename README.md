# ShareTrips — app for creating and sharing trips. 


## Structure
### Backend
- Django — backend
- Supabase — database
- Redis — cache
- Azure Storage — image storage
- OpenAI — AI service
- Pexels — stock images
- Loops — email service

Env file: 
```
SECRET_KEY=

DATABASE_USER=
DATABASE_PASSWORD=
DATABASE_HOST=
DATABASE_PORT=

AZURE_ACCOUNT_NAME=
AZURE_ACCOUNT_KEY=
REDIS_URL=
OPENAI_API_KEY=
PEXELS_API_KEY=
LOOPS_API_KEY = 
LOGIN = 
PASSWORD = 
OTP_TRANSACTION_ID = 
FRONTEND_URL = 
FRONTEND_URL_REGISTRATION = 
```
